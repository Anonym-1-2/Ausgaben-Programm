# Ausgaben-Programm

Dieses Programm ist dafür da um Ausgaben in Sectionen zu speichern und sie aufzurufen. Das Programm speicher die Sectionen auch wenn man das Programm beendet.

## Voraussetzung
* Python 3.10 oder neuer

## Installation
Klonen Sie das Repository und installieren Sie das Skript:
'git clone https://github.com/candy-c0de/Ausgaben-Programm/ausgaben.py'

## Nutzung
Gehen Sie zum 'Ausgaben-Programm' Ordner,
starten Sie das Skript mit folgendem Befehl:
'python3 ausgaben.py'
